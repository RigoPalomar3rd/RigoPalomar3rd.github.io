Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

Body tag: Defines the document's body. Head Tag: Defines information about the document Html Tag: Defines the root of an HTML document Div Tag: Defines a section in a document Section Tag: Defines a section in a document Header Tag: Defines a header for a document or section. Main Tag: Specifies the main content of a document Article Tag: Defines an article Nav Tag: Defines navigation links Footer Tag: Defines a footer for a document or section P Tag: Defines a paragraph Aside Tag: Defines the content aside from the page content. Li Tag: Defines a list term

Incline elements

Incline elements only take up as much space on a line as they need to. They do not start new lines.
